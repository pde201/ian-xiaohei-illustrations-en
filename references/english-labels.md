# English Labels Only

This skill is English-only.

## Rules

- Write all labels in English.
- Keep labels short: usually 1-4 words.
- Prefer nouns and verb phrases over full sentences.
- Expand uncommon acronyms on first use when audience is not Engineer.

## Normalization

- Use consistent naming for repeated entities in one image.
- Avoid mixed terms for the same concept (for example, do not mix "store" and "database" for one node).
- Fix spelling and capitalization before final delivery.

## Disallowed

- Do not output non-English labels unless the user explicitly asks to break English-only policy.
- Do not mix languages in one image.
