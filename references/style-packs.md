# Style Packs

Use one style pack per image.

## xiaohei-default

- Best for sparse editorial explainers.
- Xiaohei required unless architecture readability would degrade.
- Slightly absurd metaphor, high whitespace.

## minimal-monochrome

- Black-only linework with minimal accent color.
- Best for strict docs and print-friendly assets.

## whiteboard-workshop

- Collaborative sketch feeling with quick operational annotations.
- Best for internal planning and workshop recaps.

## brand-neutral-enterprise

- Clean technical style, restrained personality.
- Best for external B2B decks and architecture communication.

## Switching Rules

- If user asks for formal/enterprise style, use brand-neutral-enterprise.
- If user asks for playful/character-led style, use xiaohei-default.
- If color constraints are strict, use minimal-monochrome.
