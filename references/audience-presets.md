# Audience Presets

Use one preset per image unless the user asks to blend.

## Exec

- Goal: strategic clarity in under 3 seconds.
- Density: low.
- Labels: 3-6 short labels, outcome-oriented.
- Tone: decision-focused, minimal jargon.

## PM

- Goal: product behavior, tradeoffs, and flow clarity.
- Density: medium.
- Labels: 5-8 labels with feature/process terms.
- Tone: concrete, operational, cross-functional.

## Engineer

- Goal: technical structure and interface understanding.
- Density: medium-high to high.
- Labels: component and protocol precise.
- Tone: exact, implementation-aware, no marketing wording.

## Customer

- Goal: explain value and process without internal complexity.
- Density: low to medium.
- Labels: plain English, avoid internal acronyms unless defined.
- Tone: practical and clear.

## Mapping Rules

- If uncertain, default to PM.
- Dense architecture diagrams usually use Engineer preset.
- Sparse Xiaohei editorial usually uses Exec or PM.
