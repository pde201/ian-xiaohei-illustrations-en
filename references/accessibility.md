# Accessibility Guardrails

Apply these checks before finalizing an image.

## Color Semantics

- One semantic meaning per color in a single image.
- Do not rely only on color to encode meaning; reinforce with position, icon, or text.

## Contrast and Legibility

- Keep label text readable against white background.
- Avoid low-contrast accent text.
- Keep labels short and spatially separated to avoid overlap.

## Density Controls

- If labels are crowded, reduce nodes or split into multiple images.
- In dense architecture mode, keep one primary path visually dominant.

## Minimum Pass Criteria

- Meaning remains clear when viewed in grayscale.
- Key warnings/blockers are recognizable without color alone.
